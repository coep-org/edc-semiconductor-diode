# Statement of Work (SoW): Automotive Voltage Conditioning Module
## Objective
Design and validate a dual-stage circuit to condition a noisy ±12V automotive sensor input line down to a safe 0V–5V microcontroller range, and implement a stable local 3.3V power rail.

## Acceptable Hardware Boundaries
- Target Overvoltage Protection Limit: 5.1V Max
- Target Forward Voltage Drop ($V_f$) for Clipping: < 0.4V
- Regulator Stability: 3.3V ± 5% under load variations.
